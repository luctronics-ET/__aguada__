# Pasta de Arquivos Obsoletos

Esta pasta contém arquivos que foram identificados como obsoletos, redundantes ou não utilizados no projeto AGUADA.

## Estrutura

- `backups/` - Arquivos de backup (.backup, .old)
- `obsolete_firmware/` - Firmware antigo/obsoleto
- `old_tests/` - Testes antigos não mais utilizados
- `redundant_docs/` - Documentação redundante ou obsoleta
- `old_workspace/` - Arquivos de workspace e documentação antiga
- `logs/` - Logs antigos

## Observação

**ATENÇÃO:** Antes de deletar permanentemente, verifique se algum arquivo aqui não é necessário.

Os arquivos podem ser deletados após confirmação de que não são mais necessários.
